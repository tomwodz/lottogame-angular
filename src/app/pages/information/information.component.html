Opis

Aplikacja webowa na wzór gry lotto, oparta na modularnym monolicie, architekturze heksagonalnej. Użytkownik otrzymuje unikalny bilet po podaniu zestawu liczb. Wygrywające liczby pobierane są automatycznie i cyklicznie z zewnętrznego serwera. Następnie na ich podstawie i biletów automatycznie generowane są wyniki.

Technologie

Java 17, Maven, Git, Spring Boot: Web (RestControllers), Data, MongoDB, Testy Jednostkowe (JUnit5, Mockito, AssertJ), Testy Integracyjne SpringBootTest, MockMvc, Test Containers, WireMock, RestTemplate, Angular, HTML5, CSS
